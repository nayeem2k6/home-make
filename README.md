
HomeNest is a modern real-estate listing platform where property owners can post listings and buyers/renters can browse, review, and contact sellers. This README contains a quick overview, features, and setup instructions for the client-side of the project.


Responsive property listings — Browse properties with card/grid views and full detail pages.

User authentication & profiles — Owners can sign up, sign in, and manage their properties; users can save favorites.

Add / Edit / Delete properties — Owners can create property posts with images, price, location, and amenities.

Reviews & Ratings — Users can leave star ratings and short textual reviews; reviews include reviewer name, date, and thumbnail.

Search & Filters — Search properties by location, price range, property type, and number of bedrooms.

Image slider & gallery — Each property detail has an image carousel and thumbnails for a better browsing experience.

Admin / Owner dashboard — Owners see their listed properties, bookings, and review moderation tools.

Tech Stack

Client: React, Tailwind CSS, React Router, Axios, react-hook-form, react-icons

Server: Node.js, Express (separate repository)

Database: MongoDB (separate server)

Deployment: Vercel / Netlify for client, Heroku / Render / DigitalOcean for server (optional)
